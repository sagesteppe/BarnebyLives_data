# BarnebyLives_data
Data for the R package BarnebyLives. 

This repository hosts large data files for the [BarnebyLives](https://sagesteppe.github.io/BarnebyLives/) R package.

### Current Release (v1.0)

- **Named_Valleys.gpkg** (2.4 MB)
  - Description: Geospatial data of around 450 Basin and Range Valley names, based on multiple cartographic sources and HUC12 geometryies. 
  - Download: See [Releases](https://github.com/sagesteppe/BarnebyLives_data/releases)

### Direct download
```bash
# Using wget
wget https://github.com/sagesteppe/BarnebyLives_data/releases/download/v1.0/Named_Valleys.gpkg

# Using curl
curl -L -O https://github.com/sagesteppe/BarnebyLives_data/releases/download/v1.0/Named_Valleys.gpkg
```
